Selecting with JQuery

You can use the $ sign elements from the DOM! It will free a lot like selecting using querySelectorAll()

For Example

  • Selecting all a tags:
    • $('a')
  • selecting all elements from the class "container"
    • $(.container)
  • Selecting all elements with id "special"
    • $(#special)
