You will use the Javascript file to continually change the size of the header , but first we need to learn how to use Javascript to actually grab the HTML object from the DOM.

Here are some important document attributes:

  • document.URL--Thsi is the actual URL of the website.
  • document.body--This is the everything inside of the body
  • document.head--This is the everything in the head of the page
  • document.links--These are all the links on the page

Then we also have methods we can use to grab HTML elements:

  • document.getElementById()--Returns the element with the id
  • document.getElementByClassName()--Returns list of all elements with the tag.
  • document.getElementsByTagName()--Returns list of all elements with the tag.
  • document.querySelector()--Returns the first object matching the CSS style selector.

The first three of these are pretty self explanatory.The querySelector() can accept CSS style selector, so you can use # or,if you want ids or classes.Basically a more generalised method where you can type any CSS style.The difference between querySelector() and querySelectorAll() is that querySelectorAll will return a list of all matches, querySelector() just return the firstmatch.
